# Why Linear Regression Was Chosen
Linear Regression is simple and easy to understand.
It works well for predicting continuous numerical values like death rates.
It helps identify relationships between features such as country, year, sex, and causes of death.
It is commonly used as a baseline model in machine learning projects.

# How Linear Regression Works
Linear Regression finds the best-fit line between input features and the target value.

It predicts the output using this equation:

y=b0+ b1x1 + b2x2 + ⋯+bnxn

# Why Random Forest Was Chosen
Random Forest gives higher accuracy for complex datasets.
It handles large datasets and nonlinear relationships effectively.
It reduces overfitting compared to a single decision tree.
It works well with both numerical and categorical data.

# How Random Forest Works
Random Forest creates many decision trees using random samples of the dataset.
Each tree makes a prediction, then the algorithm combines all predictions to produce the final result.
For regression tasks:
The final prediction is the average of all tree predictions.
This improves accuracy and stability of the model.
