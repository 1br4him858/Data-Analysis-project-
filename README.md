# Why Linear Regression Was Chosen
Linear Regression is simple and easy to understand.
It works well for predicting continuous numerical values like death rates.
It helps identify relationships between features such as country, year, sex, and causes of death.
It is commonly used as a baseline model in machine learning projects.

# How Linear Regression Works
Linear Regression finds the best-fit line between input features and the target value.

It predicts the output using this equation:

y=b0+ b1x1 + b2x2 + ⋯+bnxn
